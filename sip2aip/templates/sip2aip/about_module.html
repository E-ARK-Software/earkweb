SIP to AIP conversion

The AIP, as defined in the OAIS reference model, is an information package used to transmit archival objects into a digital repository, store the objects within the system, and retrieve objects from the system. An AIP contains both metadata that describes the structure and content of an archived essence and the actual content itself.

The SIP to AIP conversion is a set of tasks that can be performed to convert an E-ARK SIP – which means that it must comply with structural and metadata requirements defined by the E-ARK project – to an E-ARK AIP.
