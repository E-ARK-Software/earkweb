AIP to DIP conversion

The Dissemination Information Package (DIP), as defined in the OAIS reference model, is the information package which is received by the consumer in response to a request for content.

The AIP to DIP conversion is a set of tasks that can be performed to convert E-ARK AIPs – which means that they must comply with structural and metadata requirements defined by the E-ARK project – to an E-ARK DIP.
